# Инструкция для работы в Markdown

## Выделение текста

Чтобы выделить текст курсивум, нужно обрамить его звёздочками (*) - *Курсив* 

Чтобы выделить текст полужирным, нужно обрамить его двумя звёздочками (**) - **Полужирный**
